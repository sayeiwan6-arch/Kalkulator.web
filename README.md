# Kalkulator.Web — Web-Based Calculator Application
 
A web-based calculator application supporting basic arithmetic operations, unit conversion, and calculation history — built as a team project where I served as Project Leader.
 
## 📖 About
 
Kalkulator.Web is a web-based calculator application developed by a three-member student team as part of an academic project. Beyond standard arithmetic operations, the app includes a unit conversion feature and a calculation history log, allowing users to track and revisit previous calculations.
 
As Project Leader, I was responsible for defining the application's scope based on user needs, coordinating task distribution across UI development, calculation logic, and quality assurance testing, and presenting the final product during our team's live demonstration.
 
## ✨ Features
 
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Unit conversion
- Calculation history log
- Simple, easy-to-use interface
## 🛠️ Tech Stack
 
- HTML5
- CSS3
- JavaScript

## 🚀 How to Run
 
1. Clone or download this repository
2. Open `index.html` in your web browser
3. No installation or build steps required
